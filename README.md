# 🛒 Blinkit Sales Dashboard – Power BI Project

## 📌 Overview
This project is a **Blinkit Sales Dashboard** built in **Power BI** to analyze sales performance, outlet distribution, item categories, ratings, and customer trends using interactive visualizations and data modeling techniques.

---

## 📊 Dashboard Features
- KPI Cards for:
  - Total Sales
  - Average Sales
  - Number of Items
  - Average Rating

- Visual Analysis:
  - Outlet Sales Trend
  - Sales by Outlet Size
  - Sales by Outlet Location Type
  - Sales by Item Type
  - Fat Content Analysis
  - Outlet Type Comparison

- Interactive Filters & Slicers

---

## 🛠️ Technologies Used
- Power BI
- DAX
- Power Query
- Data Modeling
- Excel / CSV Dataset

---

## 📈 Key Insights
- 💰 Total Sales: **1.20M**
- 📦 Total Items Sold: **8523**
- ⭐ Average Rating: **3.92**
- 📊 Average Sales: **140.99**
- 🏪 Tier 3 outlets generated the highest sales

## 🧩 Concepts Implemented
### Semantic Modeling
- Fact Tables
- Dimension Tables
- Relationships
- Star Schema

### Power BI Concepts
- DAX Measures
- Calculated Columns
- Data Transformation
- Incremental Refresh
- Dashboard Designing

---

## 🔄 Data Model
The dashboard follows a **Star Schema** structure:
- **Fact Table:** Sales Data
- **Dimension Tables:** Outlet, Item Type, Location, Fat Content
